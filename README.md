# 💰 Tip Calculator

A simple and modern Tip Calculator web application that helps users calculate the tip amount, total bill, and amount per person after splitting the bill. It provides a clean and user-friendly interface with quick and accurate calculations.

## 🚀 Features

- Enter bill amount
- Enter custom tip percentage
- Add the number of people sharing the bill
- Calculates:
  - Tip Amount
  - Total Bill Amount
  - Amount Per Person
- Input validation for incorrect values
- Reset button to clear all inputs and results
- Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🧮 How It Works

1. Enter the bill amount.
2. Enter the tip percentage.
3. Enter the number of people.
4. Click the **Calculate** button.
5. The application displays:
   - Tip amount
   - Total bill amount
   - Amount each person has to pay
